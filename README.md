# scribetools
Helpful tools for editing lyrics on Genius

Installation:
1. Install a script manager if you haven't already [(Violentmonkey reccomended)](https://violentmonkey.github.io/get-it/)
2. [Install the script](https://github.com/ziIIas/scribetools/raw/refs/heads/main/scribetools.user.js)

## Note

I made this script as a personal project for QOL fixes that I find useful, and while it should work for others too, there's definitely bugs. It's not intended as something every genius user ever should use as that would go bad. Feel free to open issues and report bugs and I will try to fix them as I can. The code is all in one userscript file and is messy, and I used AI to make some of script as it's not intended to be extremely organized. At some point I am planning on refactoring everything into a neat extension that will be compatible with [Jonas' extension](https://github.com/JonasWeinhold/Artwork-Extractor-for-Genius), and I will rewrite most of it then. For now, I hope you find some value out of this script ^_^
